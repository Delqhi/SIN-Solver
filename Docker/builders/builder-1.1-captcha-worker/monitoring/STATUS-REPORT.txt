IMPLEMENTATION DETAILS
================================================================================

ARCHITECTURE:
  - Prometheus/Alertmanager sends alerts to webhook adapter
  - Adapter receives JSON alerts on port 8093
  - Routes alerts by severity (critical/warning/info)
  - Converts to Rocket.Chat format
  - Sends to correct Rocket.Chat webhook URL
  - Returns 200 OK response

TECHNOLOGY STACK:
  - Python 3.14.2
  - Flask 2.3.3 (web framework)
  - Requests 2.31.0 (HTTP client)
  - python-dotenv 1.0.0 (configuration)
  - Gunicorn 21.2.0 (production WSGI server)
  - Docker (containerization)

ALERT ROUTING:
  Critical (P1) → #alerts-critical (group_wait: 0s, repeat: 30m)
  Warning  (P2) → #alerts-warning  (group_wait: 10s, repeat: 1h)
  Info     (P3) → #alerts-info     (group_wait: 30s, repeat: 3h)

# Room-01 Dashboard Cockpit - Glossary

## Terminology Glossary

This document defines key terms and concepts used in the Room-01 Dashboard Cockpit.

---

## A

**Agent**
An AI-powered worker service that performs automated tasks. Examples: n8n, Steel Browser, Skyvern.

**API Brain**
The central API gateway service (room-13) that provides authentication and secrets management.

## C

**CDP (Chrome DevTools Protocol)**
Protocol used to communicate with Chrome/Chromium browsers for automation.

**Container**
A Docker container running a service in the Delqhi-Platform ecosystem.

**Cockpit**
The dashboard interface for monitoring and controlling the 26-room empire.

## D

**Docker Socket**
Unix socket at `/var/run/docker.sock` used to communicate with the Docker daemon.

**Dockerode**
Node.js library for interacting with Docker.

## G

**Glassmorphism**
UI design style featuring translucent, blurred backgrounds.

## I

**Iframe**
HTML element used to embed external tools (n8n, Steel) within the dashboard.

## R

**Room**
An infrastructure service in the Delqhi-Platform ecosystem. Examples: Postgres, Redis, Vault.

**React Query**
Data fetching and state management library used in the dashboard.

## S

**Self-Reflection**
Dashboard capability to display and introspect its own documentation and state.

**Solver**
A money-making worker service. Examples: Captcha Worker, Survey Worker.

**Delqhi-Platform**
The complete 26-room automation ecosystem.

## T

**TailwindCSS**
Utility-first CSS framework used for styling.

## W

**WebSocket**
Protocol for real-time bidirectional communication (planned for v2.1).
